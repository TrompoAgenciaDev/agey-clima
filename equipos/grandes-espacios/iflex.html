iFlex Plus DC-Inverter

Sistema ideal para edificios ya construidos. Múltiples tipos de unidades interiores. Ideal para cubrir largas distancias. Flexibilidad para agregar unidades interiores en instalaciones existentes.

  • Características
  • Especificaciones
Largas distancias entre unidades exteriores e interiores.
Mayor flexibilidad para todo tipo de construcciones.

Unidades Condensadoras

Tecnología Inverter que permite alcanzar altos niveles de eficiencia energética.
Operación estable en temperaturas extremas (-20°C a 48°C).
Múltiples unidades interiores disponibles.
Controles centralizados o individuales para cada unidad interior.
Mayor flexibilidad para realizar reformas tales como agregar unidades interiores en instalaciones existentes.

Módulos Básicos

Módulos básicos configurables de 8HP, 10HP, 12HP, 14HP y 16HP.
Permite incrementar la capacidad de a 2HP desde 8HP hasta 64HP.
Se pueden combinar hasta 4 módulos básicos.
